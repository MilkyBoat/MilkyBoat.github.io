windows + VisualStudio + opencv 环境配置

此blog编辑于2021年3月8日，具体操作可能随版本变化而变化，注意教程时效性

step1：下载opencv集成包

点击链接下载最新版本的opencv，下载好后运行，这是一个自解压文件，解压之后的opencv文件夹找个地方放好，比如D:\lib\opencv。

step2：配置环境变量

依次点击：此电脑->顶上的系统属性->右边或往下滑高级系统设置->环境变量

然后上面的用户变量或下面的系统变量任选一个，点击新建->变量名填写OPENCV，变量值为D:\lib\opencv\bulid如果你之前选择了其它目录，这里修改”opencv\bulid”之前的内容与之一致。

然后找到Path环境变量，如果之前选择在用户变量中添加，这里可能没有，需要新建该环境变量。然后点击编辑->新建，输入值%OPENCV%\x64\vc15\bin
